# Purpose
This repo serves to deploy a FLNet Client which allows to participate in the
[FLNet project](https://dev.federated-learning.net/documentation), contributing data in a privacy preserving way 
towards an independent, open source research environment with almost zero trust required towards
the FLNet Platform and Network, giving you full data governance but allowing for collaboration.

For information on how to use the client, you can checkout the [documentation](https://dev.federated-learning.net/documentation/docs/client-deployment-usage/welcome)

# How to deploy
Please refer to the relevant [documentation](https://dev.federated-learning.net/documentation/docs/client-deployment-usage/deploy-client).
